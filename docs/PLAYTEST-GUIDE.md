# Nearfolk Playtest Guide

**Before you start:** Open browser console (Cmd+Option+J) and run `localStorage.clear()` to reset all state. Then refresh.

**How to use this:** Play the game at https://nearfolk.pages.dev (or localhost:5173). Follow each section in order. Write your notes in the blanks. Be honest. "It broke" and "I felt nothing" are both valid.

## 3. Build Phase — Day 1

Try placing each piece type. Note what works and what doesn't.

### Placement basics
- [ ] Ghost preview follows your cursor?
- [ ] Ghost shows red over occupied/commons cells?
- [ ] Click places the piece?
- [ ] Piece drops in with animation (falls from above)?
- [ ] You hear a sound on placement?
- [ ] Score number changes?
- [ ] Floating +N or -N appears near the piece?

### Rotation
- [ ] R key rotates the ghost preview?
- [ ] T key rotates a placed piece?
- [ ] Resident moves to new porch side when cottage rotates?

### Undo/Redo
- [ ] Right-click removes a piece?
- [ ] Ctrl+Z undoes last placement?
- [ ] Ctrl+Shift+Z redoes?

### Camera
- [ ] Scroll zooms smoothly?
- [ ] Q/E orbits smoothly (not instant snap)?
- [ ] Camera breathes (subtle up/down)?

### Connections
- [ ] Place two cottages facing each other. Golden sightline appears?
- [ ] Place a tree between them. Sightline breaks?
- [ ] Remove the tree. Sightline returns?
- [ ] Lonely cottage (facing edge) gets a sad cloud?

### Residents
- [ ] Cottage spawns a resident (small person next to it)?
- [ ] Residents have different colors?
- [ ] Hover a cottage shows name + trait tooltip?
- [ ] Tooltip shows "Right-click to remove · T to rotate"?

### Score
- [ ] Score number is visible and readable?
- [ ] Score pops (gets bigger) when it changes?
